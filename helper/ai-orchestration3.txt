To predict a series, we will use a neural network
y = f_w^ (x^) + memory

Where y is the new state
f_w^(x^) is the current state
and the memory is the previous state

example:

y_2 = f_w^(x_1^, 0)
y_3 = f_w^(x_2^,y_2)
y_4 = f_w^(x_3^,y_3)


Reminder:

Fully connected Multi Layer Perceptron is the architecture that connects each neuron to every neuron in the next layer.

What we will do, is Add another Layer - the "Memory Layer" and connect each perceptron to the memory.

Meaning, we will take each layer in the MLP, and create a memory layer for it.

For input X^ in R^n, the network will have input from the dimension n+p, where p is the number of perceptrons in the input layer.

X_in = [X^_(t+1),p_t^p]

*The output will have a size of 2p

The architechture described is called RNN

There is another way - use only a specific number of previous steps to predict the next element in the series.

This is called a "Sliding Window" (Context window in llms)

RNN is trained the same way as MLP - forward and backwards passes

Back to RNNs:

Every neuron has independent memory

There are different activation functions.

RNN can't work with large context windows, because the gradient vanishes. RNN is good for problems where short term memory is needed.

This means that the RNN from the homework will be better in recognizing high frequency signal (Need to remember less samples to recognize the signal)

This is also relevant for words auto completion - Words that change meaning slowly (In a book where a killer is mentioned in the begginning of the book will be forgotten by the RNN by the end of the book)

But the RNN can understand "David went to the beach and then Moshe jumped over the fence" well

LSTM - Long Short Term Memory

LSTM is built from different RNN with a different activation function, each RNN captures a different frequency

Element wise product is a way to close and open components in a vector

This is importand because each component in a vector representation of text is a different attribute
